# NBA Streamlit App: NBA Statistical Analysis

## Project Overview
The goal of this project is to develop an interactive Streamlit application for data exploration. The data for this app explores trends in NBA player statistis from the 1996-97 season through the 2022-23 season. This app allows users to load, filter, and analyze a dataset through a user friendly interface.

This project seeks to create an interactive application that allows users to explore and analyze a dataset. Key elements inlcude:
 
- Loading and displaying datasets using Python
- Enabling user-driven exploration through filters like dropdowns and sliders
- Generating visualizations to identify trends in the data

---

## How to Run This Project

### 1. Clone the Repository

```bash
git clone https://github.com/cccraven07/Craven-Data-Science-Portfolio.git
```

### 2. Go to the the Project Directory

```bash
cd Craven-Data-Science-Portfolio/basic_streamlit_app
```

### 3. Ensure python packages used in this project are installed

This project uses the following packages: 
- pandas
- seaborn
- matplotlib
- streamlit

### 4. Run the Streamlit App
```bash
streamlit run main.py
```

### 5. Interact with the App
Once the app is launched in your browser, you can:

- Filter data by team and season
- Explore league wide or player specific trends over time
- Interact with visualizations to generate insights

---




This app built on Streamlit allows users to explore how the NBA has evolved since the 1996-97 season. 

Explore how the the NBA has become more international over time, the correlation between draft pick and player success, or just research your favorite player's stats!
